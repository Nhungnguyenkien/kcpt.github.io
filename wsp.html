About Web Sketchpad

Web Sketchpad is software that lets you interact with sketches from The Geometer's Sketchpad using an HTML5-enabled browser on a mobile device or desktop computer. The Geometer's Sketchpad is a Dynamic GeometryÂ® exploration environment available for Macintosh and PC computers from McGraw-Hill Education.

Technical Requirements

Web Sketchpad is designed for HTML5-ready web browsers.
  • Firefox: latest version on Windows and Mac
  • Chrome: latest version on Windows, Mac, iOS, and Android
  • Safari: version 6.1+ on Mac, iOS6+
  • IE: IE9+ on Windows
